Bookie: Sell and register ticket waves for a competition

Bookie allows you to manage bets between friends on competitions. Each user chooses an option to bet on, fills a lightning-address to receive the winning reward, and sends an amount of sats. Bookie comes with a shareable ticket scanner, which can be used to quickly find a ticket (i.e. in case of a dispute).
